## SisCOND

SISTEMA PARA GESTÃO DE CONDOMÍNIOS RESIDENCIAIS.

Desenvolvido para Internet, o aplicativo pode ser acessado por dispositivos móveis e desktop.

Possibilita o gerenciamento de condôminos, funcionários, colaboradores, fornecedores e prestadores de serviços, controle almoxarifado, contabilidade geral, registro de ocorrências, blog, reserva de dependências e muito mais.
